# Problem Statement

Students often require a simple offline calculator that performs basic mathematical operations without distractions or internet dependency. This project provides a lightweight modular Python calculator that performs all common arithmetic operations using clean, maintainable code.

# Scope
Includes:
- A menu-driven CLI calculator
- Modular functions
- Error handling
- Documentation and testing

Excludes:
- GUI
- Scientific functions
- Networks or databases

# Target Users
Students, beginners, and anyone needing a fast offline calculator.

# High-Level Features
- Menu-driven interface
- Accurate arithmetic operations
- Error handling
- Modular design
